The Grounding Switch

See how real data prevents hallucinations

The Solution: Grounding, giving an AI agent real data to reference, dramatically reduces hallucination. When Karel has access to actual transaction history, customer records, and case files, he can answer accurately. Without data, he invents confident lies.

Below, pick 3 customer questions. Toggle between "Grounded" and "Ungrounded" modes for each one. Watch how Karel's response quality changes based on whether he has real data or not.

How Grounding Prevents Hallucination

1
Real data anchors responses
When Karel has access to actual transaction records, he can reference specific details: amounts, dates, merchant names. Ungrounded, he makes up plausible-sounding numbers.
2
Explicit scope boundaries
Grounded Karel can say "I don't have prior fraud reports on file" instead of inventing one. He's forced to work only with the data he actually has.
3
Verification becomes possible
Users can verify grounded responses against the original data. Ungrounded responses can't be verified, the user has no way to know what's real.
