What changed?
Only the system prompt. The user message stayed exactly the same.
Why does it matter?
The system prompt is Karel's instructions. Change the instructions, change the behavior.
Key Insight
The system prompt is not a suggestion. It is the primary driver of your agent's behavior, personality, and constraints. A vague or missing system prompt means your agent will improvise. A clear, specific system prompt means predictable, consistent responses.
