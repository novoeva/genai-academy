The 3 Requirements for True Agentiness
1. Perception: The system takes input and understands its environment. It gathers information before acting.
2. Action: The system can affect the world through tools. It doesn't just analyze, it does.
3. Feedback Loop: The system observes the outcome of its actions and uses that to inform the next decision.
